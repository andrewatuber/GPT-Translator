How it works

  • Press Start Listening and begin speaking.
  • The app records a short clip, transcribes it using OpenAI, and translates it.
  • If you speak Language A, you get Language B. Speak Language B, you get Language A. Any other language is translated to Language B.
  • Your API key stays on your device. Use "Remember" to store it locally.
